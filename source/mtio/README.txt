-- Requirements --
Python 3.x (tested on 3.8)

-- Setup --
Run setup.bat before using any of the tools to install the needed dependencies.
Ensure the setup completed successfully by making sure there aren't any error messages.

-- 3ds Max Plugin --
See the readme in the mtmaxio folder for further instructions.

-- Tools --
The tools directory contains a handful of useful tools for manipulating supported file formats.

- mtmrlconv
  This tool can convert MRL files to YML and back. Using the associated bat file you can simply drag/drop files to convert them.

- mttexconv
  This tool can convert image files to TEX and back. Using the associated bat file you can simply drag/drop files to convert them.
